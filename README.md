# 🚀 HyperNova

> Level Up Your Learning.

HyperNova is an AI-powered learning companion designed to help students understand programming, web development, AI, and computer science concepts through interactive conversations.

## ✨ Features

- 🤖 AI-powered tutoring
- 💻 Programming assistance
- 📚 Step-by-step explanations
- 📝 Practice questions
- 🌙 Modern responsive UI
- ⚡ Powered by IBM watsonx.ai

## 🛠️ Tech Stack

- React
- Node.js
- Express
- IBM watsonx.ai
- Tailwind CSS

## 🎯 Goal

HyperNova aims to make learning easier by acting as a personal AI tutor that teaches concepts instead of simply providing answers.
